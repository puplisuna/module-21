    2. coding is a way to communicate with computers. It involves writing instructions in a programming language that the computer can understand and execute.   2. why should we do coding ?

    3. Coding is essential in today's digital world. It empowers individuals to create, innovate, and solve real-world problems.     is coding easy or hard ?

    Coding can be both easy and hard, depending on various factors such as the individual's background, the programming language being learned, and the complexity of the project. The key is to start with simple concepts and gradually build up to more complex topics.
